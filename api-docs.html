Getting Started

Our REST API allows you to generate packaging templates programmatically. Perfect for integrating into your existing workflows or building custom applications.

Base URL

https://api.papercraftpro.com/v1

Authentication

All API requests require authentication using your API key:

Authorization: Bearer YOUR_API_KEY

Generate Template

Create a new packaging template with custom dimensions and settings.

Endpoint

POST /templates/generate

Request Body

{
  "width": 2.5,
  "height": 4.25,
  "flapHeight": 1.5,
  "flapStyle": "square",
  "format": "svg"
}

Get API Key

API access is available for Pro subscribers. Contact us to get your API key.
